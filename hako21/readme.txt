 使用条件（オリジナルより引用）
―――――――――――――――――――――――――――――――――――
 箱庭諸島２＋のソースについては、自己責任で使用する限り、基本的には
 自由に利用してもらってかまいません。ただし、変更を加えた場合は、変更
 したものを配布するには、後述の「配布条件」に従って下さい。

 なお、Pos.toネットへの設置は、例え「自己責任」でも禁止致します。

 また、ゲーム画面の最上部にある、箱庭諸島スクリプト配布元へのリンク
 を外すのも禁止します。デザイン上の変更は構いませんが、必ず最上部に
 'http://www.bekkoame.ne.jp/~tokuoka/hakoniwa.html' へのリンクが
 存在し、そこでスクリプトが入手できる事がわかるようにして下さい。

 付属しているgifファイルは、小川克人氏が著作権をもっています。このゲー
 ム用以外の用途に許可なく使用するのは禁止しますし、再配布も禁止します。

―――――――――――――――――――――――――――――――――――
 配布条件（オリジナルより引用）
―――――――――――――――――――――――――――――――――――
 箱庭諸島２＋のソースを改変し、それを他人に譲渡、配布する場合には、以
 下の制約を課します。

・無料配布であること。
・ゲーム画面のトップに表示される、スクリプトの配布元／
　再配布元へのリンクを消すのを禁止すること。
　また、それ以外の改造は許可すること。
・本条件と同等に、改造したものの配布を許可すること。
・配布するページにおいて、オリジナルスクリプトの配布元として当サイトへ
  のリンクを置くこと。

・再々配布する場合もこの規定が有効になります。

 注意事項
―――――――――――――――――――――――――――――――――――
・サーバーによっては、動作しないことがあります。
　（gccがインストールされていないなど）
・他の箱庭諸島２＋からの移行の場合コンパイルし直す必要があります。
　（同じルールの場合のみログ互換はあります）

　（要望があれば改造します）

 設置方法ガイダンス
―――――――――――――――――――――――――――――――――――

1.圧縮ファイルの解凍

config.cgi …環境設定ファイル
tester.html…表示テスター
hakow.js   …箱庭諸島２１世紀描画ルーチン集
hakow.css  …箱庭諸島２１世紀スタイルシート
MakeFile   …コンパイル用プロジェクト
images     …画像ファイル格納ディレクトリ
data       …データーファイルのディレクトリ
　　　　　　（新しい島を選択したときサーバーエラーになる場合必要）
*.c，*.h   …ソースコード（省略）

―――――――――――――――――――――――――――――――――――
 基本設定 （必要最小限の設定）
 config.cgiとhakow.jsをテキストエディッタで開き、
 以下の部分を修正して下さい。
―――――――――――――――――――――――――――――――――――
config.cgi
# 共通設定
titleName=箱庭諸島２１

# 管理者用パスワード
masterPassword=0123

# 管理者用特殊パスワード
specialPassword=4567

# 設置URL
cgiURL=http://???/cgi-bin/hako21/hakow.cgi

# 画像等を置くディレクトリ
fileDir=http://???/cgi-bin/hako21/

# 掲示板
bbsURL=http://???/cgi-bin/yythread/yythread.cgi

hakow.cssは任意です。

―――――――――――――――――――――――――――――――――――

3.各ファイルのアップロード
　サーバーのディレクトリに以下の通り配置して下さい
（イメージ以外はasciiモードで転送すること）

public_html (ホームディレクトリ)
      |
      +-- hako21 /
             |    MakeFile
             |    hakow.js
             |    hakow.css
             |    config.cgi   [644]
             |    *.c
             |    *.h
             |
             +-- data / info.cgi   [644]
             |          log.cgi    [644]
             |          loghis.cgi [644]
             |          map.cgi    [644]
             |
             +-- images / black.gif …



―――――――――――――――――――――――――――――――――――

※以上の最低限の設定でとりあえず掲示板は動く様になります。
